# Task: Stress-Test UI Copy

## Purpose
This task evaluates user-facing copy for trust risk and misinterpretation.

## Scope
Applies to a defined set of UI copy or flow.

## Required Context
- Active Role: UX Trust Designer
- Language Law must be loaded

## Required Inputs
- UI copy text
- Associated user flow
- Campaign state context

## Stress-Test Procedure
The copy must be evaluated from a first-time user perspective.

The evaluation must test whether a reasonable user could:
- Believe they are buying something
- Expect immediate fulfillment
- Misunderstand escrow behavior
- Misinterpret failure or delay conditions

## Required Output
The evaluation output must include:
- Identified confusion risks
- Misinterpretation scenarios
- Required copy or flow changes
- Trust impact summary

## Completion Rule
This task is complete only when copy is validated or corrected.
