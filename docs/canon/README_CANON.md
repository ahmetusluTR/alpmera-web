# Alpmera Canon System

## Purpose
This document explains how Alpmera’s Canon System works and how contributors must use it.

The Canon System is not documentation.
It is institutional law.

All decisions, features, prompts, and AI interactions must comply with this system.

---

## What the Canon System Is

The Canon System defines:
- What Alpmera is
- How decisions are made
- Who has authority
- How conflicts are resolved

It exists to prevent:
- Trust erosion
- Scope creep
- Retail drift
- Optimism bias
- Inconsistent AI behavior

---

## Canon Layer Overview

The Canon System is hierarchical.
Higher layers override lower layers.
